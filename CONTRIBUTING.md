# How to Contribute (Indie Style)

1. Fork the repo.
2. Make your change (code, docs, 3D files, encoder improvements, whatever).
3. Open a PR with a clear title and "why this helps the mission".
4. We merge fast if it works on real hardware.
5. No corporate-speak. Be direct. Be based.

Biggest needs right now:
- Motorized XY stage control (RPi.GPIO / stepper)
- Better grid decoder + Reed-Solomon error correction
- Encoder: any file → xTool-compatible dot-grid PNG
- 3D-printable frame files

Issues and discussions are open. Let's build this together.
